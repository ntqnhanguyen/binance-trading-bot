================================================================================

QUICK START AFTER CLONE
================================================================================

1. Install dependencies:
   pip install -r requirements.txt

2. Setup environment:
   make setup-env
   # Edit .env file with your API keys

3. Test API connection:
   make test-api

4. Download historical data:
   make download-data

5. Run backtest:
   make backtest

6. Run bot:
   make run
